```bash

```

```bash
yarn start
```

## 代码格式

在 VS Code 中添加以下配置，将为你在保存文件是自动格式化代码。

```json
{
  "editor.formatOnSave": true,
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true,
    "source.fixAll.stylelint": true
  }
}
```
